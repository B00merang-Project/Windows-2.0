# Windows 2.0

A GTK theme based on Windows 2.0 appearance

![win-2.0](https://raw.githubusercontent.com/B00merang-Project/gallery/master/Windows%202.0%20(1).png)

**Maintainer :** [Elbullazul](https://github.com/Elbullazul)

**License :** GPL v3

**More info :** http://b00merang.weebly.com/windows-20.html

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines

# Windows 2.0
A GTK theme based on Windows 2.0 appearance

### Preview
![win-2.0](https://raw.githubusercontent.com/B00merang-Project/gallery/master/Windows%202.0%20(1).png)

### Supported platforms
- Any GTK-based desktop
- No shell-scpecitifc theme (yet)

### Manual installation
Go to releases, download the latest `.zip` file and extract it to the themes directory i.e. `/home/USERNAME/.themes`
